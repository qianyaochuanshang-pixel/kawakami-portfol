# システム概要

## 🔰 これは何？
PHPとMySQLを使用して作成した、在庫管理ログインシステムです。  
ユーザー登録・ログイン後に、商品の管理を行うことができます。

## 🎯 何ができる？
- ユーザー登録 / ログイン  
- 商品の追加  
- 商品ごとの在庫数変更・数量チェック  
- 最終更新日 / 更新者の確認  
- 商品の削除（論理削除）  
- 削除した商品の復元
