# システム概要

## 🔰 これは何？
PHPとMySQLを使って作成した在庫管理ログインシステムです。
ユーザー登録・ログイン後に、商品の管理ができます。

## 🎯 何ができる？
- ユーザー登録 / ログイン
- 商品の追加 / 個数変更 / 商品別数量チェック
- 最終更新日 / 更新者 チェック
- 商品の削除（論理削除）
- 削除した商品の復元
